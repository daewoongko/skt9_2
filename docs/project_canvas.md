# Project Canvas: 인지-운동 미션 게임으로 확장하기

## 1. 프로젝트 주제

본 프로젝트의 주제는 **인지와 운동을 함께하는 미션 게임**이다.

사용자는 게임처럼 주어진 미션을 수행한다.  
미션은 단순 운동만 하는 것이 아니라, 기억하기, 색깔 구분하기, 숫자 계산하기 같은 인지 과제와 의자 일어서기, 균형 잡기, 제자리 스텝 같은 운동 과제를 함께 포함한다.

## 2. 문제 정의

고령자는 신체 기능과 인지 기능이 함께 저하될 수 있다.  
따라서 운동만 하거나 인지 훈련만 하는 방식보다, 두 요소를 함께 자극하는 미션형 훈련이 필요하다.

하지만 모든 사용자에게 같은 미션을 제공하면 다음 문제가 생길 수 있다.

- 초보자에게 너무 어려운 미션이 제공될 수 있음
- 건강 이상이 있는 사용자에게 위험한 운동이 추천될 수 있음
- 인지 정확도와 반응 시간이 반영되지 않을 수 있음
- 사용자의 최근 수행 결과에 따라 난이도를 조절하기 어려움
- 개인화된 피드백이 부족함

## 3. 해결 아이디어

본 프로젝트에서는 Gemini API를 **AI 미션 코치**로 활용한다.

전체 흐름은 다음과 같다.

1. 사용자가 인지-운동 미션을 수행한다.
2. 포즈 인식 모델이 운동 수행 결과를 분석한다.
3. 인지 과제 모듈이 정확도와 반응 시간을 측정한다.
4. Gemini가 두 결과를 종합한다.
5. 다음 미션과 코치 피드백을 생성한다.

## 4. 입력 데이터

AI 미션 코치가 사용하는 입력 데이터는 다음과 같다.

| 입력 항목 | 설명 |
|---|---|
| age_group | 사용자 연령대 |
| gender | 사용자 성별 |
| user_level | 초보자/중급자 등 사용자 수준 |
| health_condition | 건강 이상 여부 |
| health_notes | 무릎 통증, 고혈압 등 주의사항 |
| current_difficulty | 현재 미션 난이도 |
| completed_count | 운동 완료 횟수 |
| target_count | 목표 운동 횟수 |
| balance_status | 균형 상태 |
| failed_reason | 실패 원인 |
| cognitive_task_type | 인지 과제 종류 |
| accuracy | 인지 과제 정확도 |
| reaction_time_sec | 반응 시간 |

## 5. 출력 데이터

Gemini는 다음 형식의 결과를 생성한다.

| 출력 항목 | 설명 |
|---|---|
| mission_name | 추천 미션 이름 |
| mission_goal | 미션 목표 |
| cognitive_task | 인지 과제 설명 |
| movement_task | 운동 과제 설명 |
| difficulty | 추천 난이도 |
| difficulty_change | 난이도 변화 이유 |
| success_condition | 성공 조건 |
| coach_feedback | 사용자에게 보여줄 피드백 |
| personalized_reason | 개인화 추천 이유 |
| safety_note | 안전 주의사항 |
| expected_effect | 기대 효과 |
